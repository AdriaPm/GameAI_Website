Game AI - Patrolling & Wandering

Project Info

  • Project Work Time: 5 hours
  • Tech Stack: Unity Engine, VS Code, Unity Assets

Features

Feature Description View
AI Wander Ghosts wander scene randomly. Direction, radius, and timing of motion initialized AI Script: Wander
AI Patrol Cop Patrols between waypoints. Waypoints are gameobjects dictated by dev. First waypoint is selected randomly. AI Script: Patrol
Spotlight Flicker Cop flashlight flickers based on ghost proximity
Ghost Alert Icon Ghost Displays alert icon based on proximity to cop agent
Camera Displays Scene features cameras for top, cop pov, and cop 3rd person. All with mouse movement for looking around
